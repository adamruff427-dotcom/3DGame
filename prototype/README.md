# 3DGame prototype

This is the first playable concept slice for an original Quake-derived game.
It deliberately tests spatial design before engine integration: a vertical loop, stacked rooms, a lift, a bridge, and enemies that make elevation tactically useful.

Open `index.html` in a browser. Move with `WASD`, aim with the mouse, fire with left click, and use `E` at the lift.

The prototype uses original placeholder geometry and colors. It does not include Quake or Doom assets.
